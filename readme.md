# Massage

Aplikacja

## Technologie

- HTML, CSS, JavaScript
- Tailwind CSS (CDN)
- daisyUI (CDN)

## Uruchomienie

Otwórz projekt w VS Code i kliknij **Go Live** (plugin Live Server).

## Podgląd na telefonie

**Warunek:** telefon i komputer muszą być w tej samej sieci Wi-Fi.

1. Uruchom **Live Server** w VS Code.
2. W CMD wpisz `ipconfig`, znajdź **IPv4 Address** (np. `10.23.96.81`).
3. Na telefonie otwórz `10.23.96.81:5500`.

> IP może się zmienić po restarcie hotspota — sprawdź ponownie przez `ipconfig`.
